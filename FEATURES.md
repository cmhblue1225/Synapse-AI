## 👤 사용자 관리

### 회원가입 및 로그인
- **이메일 기반 인증**: Supabase Auth를 통한 안전한 계정 생성
- **비밀번호 보안**: 암호화된 비밀번호 저장 및 관리
- **세션 관리**: JWT 토큰 기반 자동 로그인 유지
- **프로필 관리**: 사용자 이름, 아바터 등 개인 정보 설정

### 데이터 격리
- **Row Level Security (RLS)**: 사용자별 완전한 데이터 분리
- **개인 작업공간**: 각 사용자만 자신의 지식 노드에 접근 가능
- **안전한 파일 저장**: 사용자별 독립된 파일 저장소

## 📝 지식 노드 관리

### 노드 생성
- **다양한 노드 타입**: 지식, 개념, 사실, 질문, 아이디어, 프로젝트, 리소스, 메모
- **리치 텍스트 편집**: 마크다운 지원 텍스트 에디터
- **메타데이터**: 제목, 내용, 태그, 생성/수정 시간 자동 관리
- **버전 관리**: 노드 변경 이력 추적

### 노드 편집 및 관리
- **실시간 저장**: 자동 저장 기능으로 데이터 손실 방지
- **태그 시스템**: 다중 태그 지원으로 체계적 분류
- **상태 관리**: 활성/비활성 상태로 노드 생명주기 관리
- **검색 최적화**: 자동 인덱싱으로 빠른 검색 지원

### 노드 타입별 특성

#### 📚 지식 (Knowledge)
- **학습 내용 정리**: 책, 논문, 강의 내용 등
- **참조 링크**: 외부 리소스 연결
- **관련 개념 연결**: 자동 관계 발견 및 수동 연결

#### 💡 개념 (Concept)
- **추상적 아이디어**: 이론, 원칙, 모델 등
- **정의 및 설명**: 명확한 개념 정의
- **예시 및 응용**: 실제 적용 사례

#### ✅ 사실 (Fact)
- **객관적 정보**: 데이터, 통계, 역사적 사실
- **출처 명시**: 신뢰할 수 있는 정보원 기록
- **검증 가능성**: 팩트 체크 가능한 정보

#### ❓ 질문 (Question)
- **미해결 문제**: 추후 탐구할 주제
- **학습 질문**: 이해도 확인을 위한 질문
- **연구 주제**: 깊이 있는 탐구가 필요한 주제

## 🤖 AI 기반 파일 요약

### 자동 PDF 요약 시스템

#### Phase 1: 파일 업로드
- **드래그 앤 드롭**: 직관적인 파일 업로드
- **파일 검증**: PDF 형식 및 크기 제한 확인
- **Supabase Storage**: 안전한 클라우드 파일 저장

#### Phase 2: 텍스트 추출
- **Edge Functions**: 서버사이드에서 PDF 처리
- **pdf-parse 라이브러리**: 정확한 텍스트 추출
- **에러 처리**: 손상된 PDF 파일 대응

#### Phase 3: AI 요약 생성
- **GPT-4o-mini 모델**: 비용 효율적인 고품질 요약
- **200단어 제한**: 핵심 내용만 간결하게 정리
- **한국어 최적화**: 자연스러운 한국어 요약 생성

#### Phase 4: 요약 저장 및 표시
- **데이터베이스 저장**: `file_summaries` 테이블에 영구 저장
- **토글 UI**: 클릭 한 번으로 요약 보기/숨기기
- **재생성 기능**: 언제든지 새로운 요약 생성 가능

### 요약 품질 최적화
- **컨텍스트 최적화**: 파일 유형별 맞춤형 프롬프트
- **중요도 기반 선별**: 핵심 내용 우선 요약
- **읽기 편의성**: 문단 구분 및 포맷팅

## 🌐 지식 그래프 시각화

### D3.js 기반 인터랙티브 그래프
- **동적 시각화**: 실시간 노드 및 관계 표시
- **물리 시뮬레이션**: 자연스러운 노드 배치
- **확대/축소**: 상세 내용 확인 및 전체 구조 파악

- **노드 클릭**: 상세 정보 팝업 표시
- **관계 하이라이트**: 특정 노드의 연결 관계 강조
- **필터링**: 노드 타입별 표시/숨김

### 시각적 구분
- **색상 코딩**: 노드 타입별 다른 색상
- **크기 조절**: 중요도나 연결 수에 따른 노드 크기

## 🔍 검색 및 발견

### 하이브리드 검색 시스템

#### 텍스트 검색
- **전체 텍스트 검색**: PostgreSQL의 full-text search 활용
- **자동완성**: 입력 중 실시간 검색 제안
- **필터링**: 노드 타입, 태그, 날짜별 검색 범위 조정

#### 벡터 의미 검색
- **OpenAI Embeddings**: text-embedding-3-small 모델 사용
- **1536차원 벡터**: 고정밀 의미 표현
- **의미적 유사도**: 키워드가 달라도 의미적으로 유사한 내용 발견

### 검색 결과 최적화
- **관련도 순 정렬**: 텍스트 매칭 + 벡터 유사도 점수 결합
- **미리보기**: 검색 결과에서 관련 부분 하이라이트
- **빠른 액션**: 검색 결과에서 바로 편집, 삭제, 관계 추가

## 📁 파일 관리

### 지원 파일 형식
- **PDF**: 주요 지원 형식, 자동 텍스트 추출 및 요약
- **이미지**: JPG, PNG 등 이미지 파일 첨부 (향후 OCR 지원 예정)
- **문서**: DOCX, TXT 등 텍스트 문서 (향후 지원 예정)

### 파일 처리 워크플로우
1. **업로드**: 드래그 앤 드롭 또는 클릭 업로드
2. **검증**: 파일 형식, 크기, 내용 검증
3. **저장**: Supabase Storage에 안전한 저장
4. **처리**: 백그라운드에서 자동 텍스트 추출 및 요약
5. **연결**: 지식 노드와 파일 연결 관리

### 파일 보안
- **접근 제어**: 업로더만 파일에 접근 가능
- **바이러스 스캔**: 업로드 파일 보안 검사 (향후)
- **백업**: 자동 백업으로 파일 손실 방지

## 🔗 관계 관리

### 노드 간 관계 타입
- **관련됨 (related_to)**: 일반적인 연관 관계
- **포함함 (contains)**: 상위-하위 개념 관계
- **참조함 (references)**: 인용 및 참조 관계
- **대조됨 (contrasts)**: 대립 또는 비교 관계
- **선행함 (precedes)**: 시간적 또는 논리적 선후 관계

### 관계 가중치
- **강도 표시**: 관계의 중요도를 0.1-1.0으로 표현
- **시각적 표현**: 관계 강도에 따른 선 굵기 조절
- **자동 조정**: 사용 빈도에 따른 가중치 자동 업데이트

### 자동 관계 발견
- **AI 분석**: OpenAI를 활용한 노드 간 관계 자동 탐지
- **유사도 기반**: 벡터 임베딩 유사도를 통한 관계 제안
- **사용자 확인**: AI 제안을 사용자가 승인/거부할 수 있는 UI

### 개인화된 AI 어시스턴트
